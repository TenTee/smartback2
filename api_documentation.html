Authentification

2 endpoints
POST
/api/auth/login/
Obtenir un token JWT

Authentifie un utilisateur avec username et password, retourne les tokens JWT access et refresh.

Requête

{"username": "string", "password": "string"}

Réponse

{"access": "jwt_access_token", "refresh": "jwt_refresh_token"}
POST
/api/auth/token/refresh/
Renouveler le token JWT

Renouvelle le token d'accès en utilisant le refresh token.

Requête

{"refresh": "string"}

Réponse

{"access": "new_jwt_access_token"}
